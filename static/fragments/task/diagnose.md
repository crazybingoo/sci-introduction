# Task Fragment: Diagnose

Use this fragment when the user provides an existing Introduction and wants critique.

## Method

- Map every paragraph to a paragraph job.
- Check whether the field stake reaches the real problem quickly.
- Identify overlong setup, literature-list behavior, vague gap, delayed contribution, method overload, and unsupported claims.
- If contribution fit matters, ask for the manuscript body before final judgment.

## Output Priority

Findings first, then revision strategy. Do not rewrite unless asked.
