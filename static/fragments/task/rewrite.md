# Task Fragment: Rewrite

Use this fragment when the user provides an Introduction and wants replacement prose.

## Required Inputs

- Existing Introduction.
- Manuscript body or detailed body-level outline.
- Any text the user says must be preserved.

## Method

- Diagnose first.
- Preserve facts, scope, and required terminology.
- Compress background before adding new claims.
- Rebuild the "Here we" move so it directly answers the named gap.
- Flag unsupported claims instead of silently strengthening them.
