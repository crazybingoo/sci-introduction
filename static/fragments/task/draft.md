# Task Fragment: Draft

Use this fragment when the user wants a new Introduction.

## Required Inputs

- Manuscript body or detailed body-level outline.
- Answers to the three video-guided questions, unless the body already makes them explicit.
- Evidence classes and claim boundary.

## Method

- Build the one-line argument before writing prose.
- Choose a 4-6 paragraph funnel.
- Keep the first broad-background sentence short.
- Preview evidence by class, not by figure number.
- End with a bounded claim.

If body content is missing, ask for it and stop before drafting.
