# Task Fragment: Coach

Use this fragment when the user asks for strategy, paragraph logic, gap framing, novelty framing, or whether an Introduction plan makes sense.

## Method

- Give a structural answer without drafting full prose unless asked.
- Use the three video-guided questions to expose missing logic.
- Suggest a compact funnel and the minimum evidence needed for each paragraph.
- If the user is preparing to draft, remind them that the manuscript body should come first.
